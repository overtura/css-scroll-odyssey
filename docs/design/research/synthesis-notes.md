# Synthesis Notes

## Adopted Principles
- scroll progress를 장면 reveal에만 사용한다.
- 미지원 환경에서도 정적이지만 읽기 좋은 layout을 유지한다.
- warm accent는 scene focus에만 사용한다.
- 기본 토큰은 밝은 배경, 짙은 본문 텍스트, 투명한 흰색 card surface를 사용한다.

## Rejected Principles
- scroll hijacking과 JavaScript timeline은 제외한다.
- 고유 브랜드 gradient나 full-bleed image clone은 쓰지 않는다.
- 검은 항해 배경은 기본 방향에서 제외한다.

## Token Changes
- `design-system/base.css`에 bright editorial scroll story token을 둔다.

## Golden Screens
- Opening hero
- First scene card during reveal
- Mobile stacked chapter
